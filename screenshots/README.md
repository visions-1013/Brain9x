# 截图说明

请将游戏截图放置在此目录下，并按以下命名：

- `main-game.png` - 游戏主界面截图
- `note-mode.png` - 笔记模式截图
- `dark-mode.png` - 深色模式截图
- `victory.png` - 胜利界面截图（可选）

## 截图建议

- 分辨率：1200x800 或更高
- 格式：PNG（推荐）或 JPG
- 内容：清晰展示游戏界面
- 隐私：确保不包含个人信息

## 快速截图工具

### Windows
- 使用 `Win + Shift + S` 截图
- 使用 Snipping Tool

### macOS
- 使用 `Cmd + Shift + 4` 截图
- 使用内置截图工具

### Linux
- 使用 `gnome-screenshot`
- 使用 `scrot` 命令行工具
